Функциялар

Функция — бұл белгілі бір тапсырманы орындайтын код бөлігі. Python тілінде функция бір рет жазылады, бірақ оны бірнеше рет қолдануға болады. Бұл кодты қысқа, түсінікті және ыңғайлы етеді. Функциялар арқылы үлкен бағдарламаларды бөліктерге бөлуге болады. Функция def кілт сөзі арқылы жасалады.

Есептер


def salam():
    print("Сәлем, бұл функция!")

salam()


def kosu(a, b):
    return a + b

print(kosu(5, 7))
